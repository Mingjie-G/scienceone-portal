## ScienceOne Portal 详细结构设计要求

以 .ai 为准首屏，科技感优化中间，震撼收尾。

| 模块 | 二级模块 | 具体解释 |
|------|----------|----------|
| 欢迎页面 | 开始使用 | **Hero区域**：科技感主视觉 + 平台slogan，传达"AI驱动科研范式革命"的使命。参考OpenAI for Science的开篇语"We’re building the next great scientific instrument to accelerate discovery."，ScienceOne应有一句有力、简洁、可感知的价值主张（如："让科研发现加速N倍"）。放置「立即开始」和「了解更多」两个CTA按钮。 |
| 欢迎页面 | 列表数字展示 | **数据亮眼区**：用大数字+标签形式展示平台核心能力数据，参考OpenAI for Science的"accelerating science"叙事逻辑，展示如：已收录文献量（亿级）、覆盖学科数、平台用户量、产生的综述报告数等。数字要真实可信，既展示规模，也传递平台可信度。 |
| 欢迎页面 | 平台化智能科研范式 | **科学哲学层**：阐明ScienceOne不是单一工具，而是一种新的科研范式——从文献检索→综述写作→实验设计→成果产出的全链路AI协同。参考OpenAI for Science的"Why accelerating scientific progress matters"，说明ScienceOne如何压缩从想法到发现的时间。 |
| 核心功能矩阵 | 深度研究 | **极速模式/研究模式**：极速模式面向日常问题解答，高效简洁；研究模式（AgentLoop架构）面向复杂调研任务，多智能体协同，过程透明可追溯。工作区展示任务/信息/文件三栏，实时呈现搜索来源和执行步骤。差异化：区别于通用AI，ScienceOne深度研究深度绑定学科数据库和文献库，来源可溯。 |
| 核心功能矩阵 | 科学工具 | **工具广场**：展示精选科研工具（如PDE-Agent、科学计算器等），按学科和场景双维度分类。Banner展示热门/自研工具，提供GitHub/ArXiv入口推动开源共享。每个工具卡片含名称、学科标签、简介，点击直达使用页面，支持收藏和分享。差异化：垂直领域专业工具，非通用AI能力的简单包装。 |
| 核心功能矩阵 | 科学数据库 | **数据枢纽**：汇聚全球权威科学数据库，覆盖材料科学、生命科学、流体力学、地球科学、空间科学、神经科学、数学等学科领域。展示数据库名称、调用方式、访问地址、数据库概览；支持一键跳转官网和链接分享。差异化：科研专属，非通用搜索可比。 |
| 核心功能矩阵 | 科学文献库 | **个人文献中心**：用户自建文献库，支持批量上传和管理；可邀请团队成员协作，权限分为管理员/协作者/成员。提供全库对话、单篇精读、论文导读、研究图谱（引文网络可视化）、文献对话、批注等功能。差异化：从阅读到对话，AI与文献深度融合。 |
| 核心功能矩阵 | 科学实验站 | **计算与仿真**：提供专业科学计算环境，支持物理、化学、生物等学科的模拟与验证。对接科学工具模块，提供可重复使用的实验流程。差异化：理论→验证的闭环，而非仅停留在文本层面。 |
| 核心功能矩阵 | 科研资讯 | **信息聚合流**：聚合国内外100+学术信息源，涵盖论文、新闻、会议通知等。瀑布流卡片展示，来源渠道丰富（arXiv、科学网、中科院等）。支持内容分类筛选（论文/新闻）和时间范围筛选。差异化：AI自动摘要+来源追溯，无需逐篇手动查阅。 |
| AI科研工作流 | 选题研判 | **AI辅助选题**：基于海量文献和最新科研动态，AI帮助研究者评估选题新颖性、可行性、研究空白。可视化呈现相关领域的"研究热力图"。参考OpenAI for Science的"built with scientists"理念，选题研判由真实科研需求驱动。 |
| AI科研工作流 | 学术综述 | **智能综述引擎**：用户输入研究问题，AI自动规划综述大纲，经用户确认后全自动化执行。支持配置目标语言、文献库来源、网络信源。输出万字级完整综述，含摘要、正文、参考文献，支持PDF/Word/Markdown多格式下载。差异化：超长上下文+多源融合，非通用摘要工具可比。 |
| AI科研工作流 | 学术审稿 | **AI辅助评审**：为研究者提供论文审稿意见，涵盖研究合理性、创新性、局限性、论据充分性等方面。帮助作者在投稿前发现潜在问题，提升论文质量。参考OpenAI for Science的"test hypotheses faster"理念，将审稿周期从月级压缩到分钟级。 |
| AI科研工作流 | 科研配图 | **学术图片生成**：基于自然语言描述生成专业学术配图（分子结构图、实验流程图、数据可视化图表等）。学术风格，精准匹配研究内容，支持多轮对话优化。可点赞/点踩反馈，形成质量闭环。差异化：科研场景深度适配，非通用图片生成。 |
| AI科研工作流 | PPT/海报 | **一键学术汇报**：上传文献（PDF/DOC/DOCX），AI自动解析结构→生成大纲（可编辑）→逐页生成内容+视觉布局→编译PPTX。实时任务进度台展示全流程日志。输出专业学术汇报材料，图片图表真实可溯源。差异化：从文献到汇报材料，一站式完成，无需手动复制粘贴。 |
| 应用案例 | 真实科研故事 | **案例展示区**：展示研究者使用ScienceOne的真实成果，如：某教授用深度研究模式在3天内完成领域综述、某团队用科学工具将计算时间从周级压缩到小时级。真实案例 > 功能列表，参考OpenAI展示与UCLA教授合作案例的方式。 |
| 技术底座 | 科学模态理解 | **多模态科学AI**：展示ScienceOne对科学文献（LaTeX公式、化学结构、实验数据图）的深度理解能力。强调模型不只是读文字，而是真正理解科学表达的语义。 |
| 技术底座 | 基准表现 | **客观能力证明**：以学术Benchmark（如MMLU、Math、Science等）数据，展示ScienceOne在不同学科领域的表现。与GPT-5、Claude等通用模型对比，突出科研专项能力优势。参考OpenAI Frontiers Science评测页面，用数据说话。 |
| 生态联盟 | 高校、科研所、企业 | **生态展示**：展示与高校、科研机构、企业的合作案例，建立"学术可信"背书。合作方式包括：模型共建、定制部署、API开放、联合研究等。尾部CTA引导机构用户联系商务合作。参考OpenAI for Science的"Get involved"板块。 |
